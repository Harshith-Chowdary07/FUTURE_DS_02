 Customer Churn Analysis Dashboard 

## 🧾 Overview

This project presents a comprehensive **Customer Churn Analysis Dashboard** built using **Power BI**. The goal of this project is to analyze customer data, identify churn patterns, and generate meaningful business insights that help organizations improve customer retention.

Customer churn is a critical problem for subscription-based businesses. This dashboard provides a data-driven approach to understanding why customers leave and how to reduce churn.

## Project Objectives

* Analyze customer behavior and service usage
* Identify key factors contributing to churn
* Calculate and visualize churn rate
* Provide actionable insights for improving retention
* Build an interactive and visually appealing dashboard

##  Dataset Information

The dataset used in this project is the **Telco Customer Churn Dataset**, which includes:

* Customer demographics (Gender, Senior Citizen, Dependents)
* Account information (Tenure, Contract type)
* Services subscribed (Internet, Phone, Streaming, etc.)
* Billing details (Monthly Charges, Total Charges, Payment Method)
* Churn status (Yes / No)

##  Tools & Technologies

* **Power BI Desktop** – Dashboard development
* **Power Query Editor** – Data cleaning & transformation
* **DAX (Data Analysis Expressions)** – Measures & calculations
* **Data Visualization Techniques** – Interactive charts & KPIs

## Data Preparation Steps

1. Imported CSV dataset into Power BI
2. Cleaned and transformed data using Power Query:

   * Handled missing values
   * Corrected data types
   * Removed inconsistencies
3. Created calculated columns and measures using DAX

##  Key Metrics & Calculations

### Total Customers

Total number of customers in dataset

### Churn Customers

Number of customers who left

### Churn Rate

Churn Rate = DIVIDE([Churn Customers], [Total Customers]) * 100

### Average Monthly Charges

Average of monthly billing amount

### Average Total Charges

Overall customer revenue contribution

## Key Insights

* Customers with **month-to-month contracts** have higher churn rates
* **Fiber optic users** show higher churn compared to DSL users
* Lack of **security and tech support services** increases churn probability
* Customers with **short tenure (0–2 years)** are more likely to churn
* Payment methods like **electronic check** show higher churn
